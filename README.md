## PRD 价值主张设计 15%

### PRD1.加值宣言 3%
许多用户对于各种车辆的了解不太熟悉，可能只知道汽车品牌，而不了解每个系列的特色或价格差异，而我们团队设想的这个软件功能是可以令用户随时随地在路边看到感兴趣的车，只需要拍照上传到该软件，系统就会进行搜索并列出详细信息；另外，该软件还涵盖了汽车证件文字识别的功能，方便客户保存关于自己汽车的相关资料，有需要处理相关事务时能够随时查看。
